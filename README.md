SmartMart is a simple product catalog web application built with Next.js and
Express.js.  
The project demonstrates public and protected routes, mock authentication, and
product management functionality.

---

## Implemented Features

- Landing page with 7 sections
- Sticky navigation bar
- Public product listing and details page
- Mock authentication using cookies
- Protected route using Next.js middleware
- Add new product functionality
- Express.js REST API
- Toast notification on successful product creation

## Feature Explanation

### Authentication:

- A mock login system using hardcoded credentials. Authentication state is
  stored in cookies.

### Protected Routes:

- Next.js middleware is used to restrict access to the Add Product page.

### Product Management:

- Products are fetched and stored using an Express.js backend API.

### UI & UX:

- Tailwind CSS is used for styling and React Hot Toast provides feedback on
  actions.
